# Install broask

## Homebrew

```bash
brew tap mulhamna/tap
brew install broask
```

Or in one command:

```bash
brew install mulhamna/tap/broask
```

## Scoop

```powershell
scoop bucket add mulhamna https://github.com/mulhamna/scoop-bucket
scoop install broask
```

## Go

```bash
go install github.com/mulhamna/broask@latest
```

## Manual install

Download the binary for your platform from GitHub Releases and add it to your `PATH`.

## Verify

```bash
broask version
```
